# koa-next-api

### Node required

**node version >= 6.0**



### For development

```bash
  # install eslint (optional)
  $npm install -g eslint eslint-plugin-standard eslint-config-standard eslint-plugin-promise babel-eslint

  # install packages
  $npm install

  # install nodemon
  $npm install -g nodemon

  # run and watch
  $npm run watch

  # build
  $npm run build
```

Recommend use [cnpm](https://cnpmjs.org/)



### For production

```bash
  # start
  $npm start
```
