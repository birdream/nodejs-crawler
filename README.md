# nodejs-crawler
nodejs-koa2-爬虫练习


### Node required

**node version >= 6.0**



### For development

```bash
  # install eslint (optional)
  $npm install -g eslint eslint-plugin-standard eslint-config-standard eslint-plugin-promise babel-eslint

  # install packages
  $npm install

  # install nodemon
  $npm install -g nodemon

  # run and watch
  $npm run watch

  # build
  $npm run build
```
